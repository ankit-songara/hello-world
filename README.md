# Hello World

A simple Go program that prints "Hello, World!" to the console.

## Overview

This is a basic "Hello World" example in Go, perfect for beginners learning the language.

## Getting Started

### Prerequisites

- Go 1.16 or higher

### Installation

Clone the repository:
```bash
git clone https://github.com/ankit-songara/hello-world.git
cd hello-world
```

### Running

```bash
go run main.go
```

**Output:**
```
Hello, World!
```

## Project Structure

- `main.go` - Entry point of the application

## What You'll Learn

- Basic Go syntax and structure
- The `main()` function as entry point
- Using the `fmt` package for printing output
- Package declarations
